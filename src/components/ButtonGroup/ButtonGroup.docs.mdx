# ButtonGroup

Visually joins a row (or column) of `Button`s into a single connected
control — the shared `role="group"` container removes rounding between
adjacent buttons and collapses their shared border, while keeping the
first/last child rounded on the outer edge only.

## Composition

`ButtonGroup` is layout-only: it doesn't clone or configure its children,
so any interactive element (not just `Button`) can be placed inside it.
Each child keeps its own `hover`/`focus-visible` state raised above its
neighbors via `z-10`, so overlapping borders don't clip a focus ring or
hover background.

## Overriding styles

Like every other component in this library, `ButtonGroup` resolves its
`className` prop through `cn()`, so an app-level class always wins over
the defaults:

```tsx
<ButtonGroup className="shadow-z3">
  <Button variant="secondary">One</Button>
  <Button variant="secondary">Two</Button>
</ButtonGroup>
```
