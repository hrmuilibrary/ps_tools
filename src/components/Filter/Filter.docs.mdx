# Filter

A pill-shaped trigger for a filter dropdown. Shows a `label`, an optional
selected `value` (rendered after a dot separator), and toggles an
absolutely-positioned popover of `children` when clicked. Closes on any
outside click.

## Active state

`Filter` is visually "active" (brand-tinted pill) whenever `active` is
`true` **or** `value` is defined — so passing any `value`, including an
empty string, marks the filter as active without needing a separate flag.

## Clearing a value

When the filter is active and `onClear` is provided, the trailing chevron
is replaced by a clear (×) affordance that stops the click from also
toggling the popover open. Omit `onClear` to always show the chevron.

## Popover content

`children` renders inside an absolutely-positioned panel below the trigger,
shown only while the filter is open. Compose it with any content —
checkboxes, radio groups, a date range, or a custom form.

## Composition

`Filter` forwards its ref and spreads the remaining native `<div>` props
onto its root element, so it composes with layout utilities and other
components in this library without a separate wrapper.
